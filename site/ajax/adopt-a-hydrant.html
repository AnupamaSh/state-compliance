Adopt-a-Hydrant
About Adopt-a-Hydrant
Adopt-a-Hydrant is a web application that allows citizens to "adopt" civic infrastructure, such as fire hydrants that need to be shoveled out after it snows.

Historical Information
The project started in 2011 when Erik Michael-Ober was a Code for America Fellow, partnered with the City of Boston.
